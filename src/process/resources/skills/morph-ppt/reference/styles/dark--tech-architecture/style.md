# 09-tech-architecture — 技术架构分享


## 风格概述
深色技术风格，适合架构分享、技术培训、系统设计演示

- **场景**: 技术分享、架构评审、团队培训、技术路演
- **情绪**: 专业、技术、清晰、现代
- **色调**: 深色、科技蓝、紫、青
- **行业**: 软件、互联网、云计算、AI、SaaS
