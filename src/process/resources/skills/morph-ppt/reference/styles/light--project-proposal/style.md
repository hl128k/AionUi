# 12-project-proposal — 项目提案方案


## 风格概述
浅灰蓝深蓝金色专业配色，适合项目立项、商务提案、方案汇报等专业场合

- **场景**: 项目立项、商务提案、方案汇报、投标答辩
- **情绪**: 专业、可信、高效、严谨
- **色调**: 冷色调、低饱和度、商务灰蓝
- **行业**: 咨询服务、科技企业、金融投资、政府项目

## 设计手法

- 浅灰蓝深蓝金色专业配色
- 专业文档布局
- 信息卡片展示
- 数据可视化图表
- 横向时间轴
- Morph过渡动画
- 风险分析展示
- 坐标冲突已修复
- 内容卡片视觉层次增强
