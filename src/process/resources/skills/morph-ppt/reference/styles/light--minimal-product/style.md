# 05-minimal-product — 极简产品介绍


## 风格概述
浅灰背景深灰主色绿色强调的极简主义风格，采用中央聚焦+极简留白布局，适合产品发布、科技展示、商务演示等场合

- **场景**: 产品发布、科技展示、品牌介绍、商务演示
- **情绪**: 专业、现代、简约、高端、科技
- **色调**: 冷色调、低饱和度、高对比
- **行业**: 科技、电子、软件、互联网、金融

## 设计手法

- 浅灰背景深灰主色绿色强调
- 中央聚焦布局
- 极简留白设计
- 细线条装饰
- 高对比度设计
- Morph过渡动画
- 装饰元素规范化
